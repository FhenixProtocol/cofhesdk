# Cofhe SDK

Build privacy-preserving dApps on Fhenix using fully homomorphic encryption (FHE). This site is organized by **what you’re building** (SDK usage, Hardhat workflows, reference material), so you can jump straight to the right place.

## Where to start

- First time using the SDK: [Quick Start](/quick-start)
- Want the full mental model first: [Introduction](/intro)
- Coming from `cofhejs`: [Migrating from `cofhejs`](/migrating-from-cofhejs)
